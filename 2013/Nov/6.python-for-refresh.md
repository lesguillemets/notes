Python の for のアレのアレとか
----------------------------------

アレのアレ，全く要領を得ないが．要するに `for i in a:` とかやるときにどっかのタイミングで
 `a` が固定されたりするのかなー的な．

```python
#!/usr/bin/python

a = range(10)
i = 0
for n in a:
    print n,
    try:
        a[i+1] *= 3
    except IndexError:
        pass
    i += 1
print a
```

で出力は

```
0 3 6 9 12 15 18 21 24 27 [0, 3, 6, 9, 12, 15, 18, 21, 24, 27]
```

なるほど．で，あれば

```python
#!/usr/bin/python

a = range(10)
i = 0
for n in list(a):
    print n,
    try:
        a[i+1] *= 3
    except IndexError:
        pass
    i += 1
print a
```

(`list(a)` に変更)

```
0 3 6 9 12 15 18 21 24 27 [0, 3, 6, 9, 12, 15, 18, 21, 24, 27]
```

ならば

```python
#!/usr/bin/python

a = range(10)

for (i,n) in enumerate(a):
    print n,
    try:
        a[i+1] *= 3
    except IndexError:
        pass
print a
```

では

```
0 3 6 9 12 15 18 21 24 27 [0, 3, 6, 9, 12, 15, 18, 21, 24, 27]
```

たとえばここで `a.append(None)` とか入れるとぐんぐん伸びるわけですねー．

こうすると

```python
#!/usr/bin/python

a = range(10)
b = enumerate(a)

for (i,n) in b:
    print n,
    try:
        a[i+1] *= 3
    except IndexError:
        pass
print a
```

やっぱり

```
0 3 6 9 12 15 18 21 24 27 [0, 3, 6, 9, 12, 15, 18, 21, 24, 27]
```
